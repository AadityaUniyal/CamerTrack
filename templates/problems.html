üì∑ Camera Issues
‚óè High Camera feed not showing ‚Äî black screen

This usually means the camera index is wrong or camera access is blocked.

Steps to fix:

  1. Make sure no other application (Zoom, Teams, OBS) is using the camera.
  2. Try switching camera index using the Cam 0 / Cam 1 / Cam 2 buttons in the dashboard.
  3. Check your OS camera permissions for Python/Browser.
  4. Restart the Flask server and refresh the page.
python app.py # restart server
üí° Tip: On Windows, go to Settings ‚Üí Privacy ‚Üí Camera and allow access.
‚óè Medium Low FPS / laggy video feed

Running many AI modules simultaneously is CPU-intensive. High resolution also adds overhead.

Solutions:

  • Disable modules you are not actively using ‚Äî each one consumes CPU cycles.
  • Emotion detection and Age/Gender are the heaviest modules; disable them first.
  • Lower the camera resolution in config.py if available.
  • Close background applications consuming CPU.
üí° Tip: Running 3‚Äì4 modules simultaneously is the sweet spot for smooth 25+ FPS.
üß† AI Module Issues
‚óè Medium Emotion detection is slow or freezes

DeepFace loads a large neural network model on first activation. This is expected behavior.

  • First activation takes 5‚Äì15 seconds to load the model into memory.
  • Subsequent frames will run faster after the initial warmup.
  • Ensure deepface and tensorflow packages are installed.
pip install deepface tensorflow
‚óè Medium Object detection not detecting anything

MobileNet SSD requires the model weight files to be present in the models/ directory.

  • Check that models/ contains the required .prototxt and .caffemodel files.
  • Make sure the object is within the 20 COCO classes (person, car, bottle, etc.).
  • Increase room lighting ‚Äî detection accuracy drops in low light.
üí° Tip: The confidence threshold is set to 0.5 by default. Objects with lower certainty are not shown.
‚óè Low QR scanner not reading codes

QR scanning accuracy depends on lighting and code size in frame.

  • Hold the QR code at least 15‚Äì20 cm from the camera.
  • Ensure even, bright lighting ‚Äî avoid glare on glossy QR codes.
  • Install pyzbar: pip install pyzbar
  • On Windows, you may also need to install the zbar DLL.
pip install pyzbar opencv-python
üì¶ Installation & Setup
‚óè High ModuleNotFoundError when starting the server

A required Python package is not installed.

pip install -r requirements.txt

If the error persists, install the missing package directly:

pip install flask opencv-python mediapipe deepface numpy
üí° Tip: Use a virtual environment to avoid package conflicts: python -m venv venv
‚óè Low Port 5000 already in use

Another process is occupying port 5000. Change the port in config.py:

PORT = 5001 # change to any free port

Or kill the process using that port:

netstat -ano | findstr :5000 taskkill /PID [PID_NUMBER] /F
‚ùì General FAQ
‚óè Info Can I use this without a GPU?

Yes! Most modules run fine on CPU. Only Age/Gender and Emotion Detection benefit significantly from a GPU. All other modules (Face Detection, Hand Tracking, Pose Estimation, QR Scanner, etc.) run smoothly on CPU-only machines.

‚óè Info How do I add a new user account?

Use the Sign Up tab on the login page to create a new account. Accounts are stored in memory during the session ‚Äî they reset when the server restarts. To persist users, add them to the USERS dict in config.py.

‚óè Info Where are screenshots and recordings saved?

Screenshots are sent directly to your browser as a download. Recordings are saved to the project directory on the server. Check the terminal output for the exact file path after stopping a recording.
