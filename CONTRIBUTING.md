# Contributing

Thanks for your interest in contributing to AI Vision Studio.

## Getting Started

1. Fork the repository
2. Clone your fork: `git clone https://github.com/<your-username>/CamerTrack.git`
3. Create a branch: `git checkout -b feature/your-feature-name`
4. Make your changes, commit, and push
5. Open a Pull Request against `main`

## Code Style

- Follow PEP 8 for Python code
- Keep functions focused and short
- No hardcoded credentials or file paths

## Reporting Bugs

Open an issue and include:
- Your OS and Python version
- Steps to reproduce the problem
- Any error messages or tracebacks

## Feature Requests

Open an issue describing the feature and why it would be useful.
